## 2024-05-24 - Accessibility Standard for Dynamic List Items
**Learning:** For interactive elements in lists (like icon buttons), providing a dynamic `contentDescription` that includes the item's unique identifier (e.g., student name) significantly improves accessibility over a static label.
**Action:** Always prefer dynamic, personalized `android:contentDescription` labels for list item actions (e.g., 'Send WhatsApp reminder to [Name]') rather than generic ones.
